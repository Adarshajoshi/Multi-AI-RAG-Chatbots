# MultiAI RAG

A lightweight **Retrieval-Augmented Generation (RAG)** project built with **LangChain** and open/free language models.

This project demonstrates how to:
- Load and structure documents
- Generate embeddings
- Store vectors in a vector database
- Retrieve relevant context
- Answer questions using an LLM
